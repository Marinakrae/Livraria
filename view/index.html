Olá!

Este sistema foi desenvolvido com o intuíto de realizar uma tarefa da universidade de tema livre. Escolhi essa temática pensando em um site em que minha namorada pudesse cadastrar seus livros favoritos como forma de recomendação para mim. Até o momento o sistema está incompleto e com algumas alterações a serem feitas. Uma delas é corrigir a opção de update de livros. Por enquanto, não é possível alterar seu gênero pois o tipo de select que eu estou usando, não sei como vou fazer para torna-lo dinâmico no caso de edição. Além disso, quero adicionar opções de multiplos gêneros. Os próximos objetivos são adicionar uma opção de feedback sobre os livros e uma nota de avaliação. Como adições futuras, caso continue desenvolvendo, planejo criar opções para filmes e séries, e aumentar os campos descritivos para livros.
